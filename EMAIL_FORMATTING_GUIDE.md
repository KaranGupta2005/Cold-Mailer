# 📧 Email Formatting Guide

## How to Format Your Email Content

The email template now supports rich formatting including tables, links, bold text, and more!

---

## 1. Creating Links

Use markdown-style links:

```
[Register Here](https://example.com/register)
[Visit Website](https://invictusdtu.in)
```

**Result:** Clickable blue links in the email

---

## 2. Creating Tables

Use pipe `|` characters to create tables:

```
Category | Event Name | Focus Area | Action Link
--- | --- | --- | ---
Hackathons & AI | Cryptoforge | Web3 & Blockchain | [Register Here](https://link.com)
Coding & CS | Coderush | Competitive Programming | [Register Here](https://link.com)
Robotics | Robo Soccer | Autonomous 2v2 Match | [Register Here](https://link.com)
```

**Result:** Professional HTML table with alternating row colors

---

## 3. Bold Text

Use double asterisks:

```
**Important:** This is bold text
**Prize Pool:** Win cash prizes
```

**Result:** Bold, emphasized text

---

## 4. Italic Text

Use single asterisks:

```
*Note:* This is italic
*Dates:* 27th Feb - 1st March
```

**Result:** Italic text

---

## 5. Section Headings

### Numbered Headings
```
1. Featured Tech & Coding Arenas
2. Featured Non-Tech & Management Arenas
3. Featured Core Engineering Arenas
```

**Result:** Large blue headings

### Regular Headings
End with a colon:
```
Why You Should Participate:
Event Logistics:
Stay Connected:
```

**Result:** Medium-sized bold headings

---

## 6. Paragraphs

Separate paragraphs with blank lines:

```
This is the first paragraph.

This is the second paragraph.

This is the third paragraph.
```

## Tips for Best Results

1. **Use blank lines** between sections
2. **Keep table columns aligned** with pipes `|`
3. **Use markdown links** `[text](url)` for clickable links
4. **Bold important info** with `**text**`
5. **Number your sections** for better organization
6. **Test with 1-2 emails** before sending to full list
